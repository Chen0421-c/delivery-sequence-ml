# Research Framework

## Working title
Road Freight Delivery Sequence Optimization using Machine Learning

## Core idea
Use historical last-mile delivery route data to train a machine learning model that predicts stop visiting sequences, then compare the result with traditional baseline methods.

## Architecture
Raw Dataset -> Memory-safe Data Loading -> Preprocessing -> Feature Engineering -> Baseline Sequencing -> ML Model -> Sequence Generation -> Evaluation -> Visualization Prototype
